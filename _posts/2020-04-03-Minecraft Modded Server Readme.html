Introduction

Minecraft is an open world sandbox developed by Mojang. One of the defining features of Minecraft is the ability for players to create their own content known as mods that fundamentally change how the game is played. This guide will be showing you not only how to take advantage of these mods but also how to play with others while using them by teaching you how to host the game from your own computer.

Requirements

  • A copy of Minecraft and its game launcher which can be acquired here and the requirements to run it.

  • At least 8 GB of RAM but recommended is 16 GB - RAM is the biggest factor in smoothly running a modded server. A modded server for a few people can easily take up to 4 GB of RAM but is largely dependent on what mods you chose and how intensive they are. You can check how much RAM you have on windows by pressing the Windows key, searching Control Panel, clicking on System and Security, and then System.

  • Computer Specs that are capable of running the mods of your choosing. - While most mods will only need the requirements to run the base game of Minecraft some mods, like shaders for example, may require more powerful PC parts. For these mods be sure to check the specific requirements on their download page.

  • LogMeIn Hamachi - This is a program used to connect other people to your server and is a alternative to port forwarding which will not be explained in this guide. It can be downloaded here.

Installation

Modding your Minecraft Client

Step 1. Choose a version of Minecraft.

This can be decided based on what mods you want to install. Ideally you want to find a version that all of the mods work on while also having that version be as new as possible. A popular website to download mods from is Curse Forge.

Step 2. Install Forge.

Once you have decided your version you’re going to go to Minecraft Forge, find the version you selected, and download the latest version Windows Installer. When the download finishes open the file and you will be greeted with this window.
Be sure to select Install Client and alter the directory if you like then select OK.

Step 3. Open Minecraft Forge version.

Now that forge has installed, open your Minecraft client, click the drop down button next to the PLAY button and you should see a version named Forge. Click on that version and run the game.

Step 4. Have all other players replicate this process.

You have successfully modded your Minecraft. Every person that wants to play with you will need to replicate this process so feel free to direct them to this guide.

Setting up the Modded Server

Step 1. Install Forge server.

Reopen the Forge installer you downloaded for the previous section but this time you’re going to select Install Server. For the directory I would recommend you pick somewhere easy to reach like a folder on your desktop.

Step 2. Run server file.

After the server installs into the folder you should see a executable named minecraft_server.*the version of forge you downloaded*. Run this file.

Step 3. Agree to Forge eula

After running the file some additional files and folders will appear inside your server folder. Open the text file named Eula and change the line eula=false to eula=true.

Step 4. Create the server mod folder.

Run the file named Forge-*version number*. You will see more folders appear, most notably the mods folder. This is where all mods you want running on your server will go.

Step 5. Run the server.

Finally run the minecraft_server.*version number* once again. This time it will open up a console with RAM usage and a command line and begin starting the server. Once you see the message World Loaded you have successfully created a modded minecraft server.

Usage

Starting the server and Joining

Step 1. Add your mods.

When the game opens you will see that the menu is slightly different from the usual screen. Most notably is the mods button now located on the menu. Click that button to see all the mods you have installed. From that menu you can also open the mods folder which is where all your downloaded mods will need to be placed. Most will come in the form of a .jar file which can just be dragged and dropped into the folder.

Step 2. Check mod compatibility.

Make sure all the mods you want are compatible by restarting the game. If there is an error with any of the mods a message will be displayed stating what is missing or the error may be.

Step 3. Copy mods to the server.

Once your Minecraft loads without errors you can then copy and paste all mods from your Minecraft mod folder to the server mod folder.

Step 4. Launch the server.

Launch the server using the minecraft_server.*version* file.

Step 5. Use Hamachi to allow others to connect.

Open LogMeIn Hamachi and log in. Click the power button and under network press create a new network and have the other players join the network. From there all players can right click and click on your network and copy IPv4 address.

Step 6. Join the server.

Once the server is running all players can go under the multiplayer menu in Minecraft and Direct Join with the address you copied earlier.

Troubleshooting

  • If any of the files in the server folder don’t open or open and immediately close make sure that you have the most recent version of Java as having an older version of Java can cause improper loading.
  • Many errors that are reported upon loading the game are due to certain mods you have downloaded needing other mods to work correctly. Make sure you install all required mods found on the target mod’s page.
  • This method of creating a server is also based on establishing a connection between you and the other players so if your internet is having issues it can affect other players connection in the game.
